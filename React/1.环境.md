# 介绍

[文档](https://doc.react-china.org/docs/installation.html#%E5%B0%9D%E8%AF%95-react)

## 安装

```shell

npm install -g create-react-app
create-react-app my-app

cd my-app
npm start
```

敲完最后一步命令后会跳出欢迎网站
